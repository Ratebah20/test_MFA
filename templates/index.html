Bienvenue sur la simulation Selfcare

Démonstration de l'intégration OTP

Cette application simule l'intégration de Selfcare avec le système MFA d'Orange Luxembourg via la méthode OTP.

Fonctionnalités disponibles :

  • Réception et validation des tokens OTP
  • Simulation du dashboard après authentification
  • Cas particulier : Première connexion avec changement de mot de passe
  • Simulation des scénarios d'erreur (token invalide, expiré, etc.)

Pour commencer

Pour tester l'intégration, vous avez deux options :

Simuler une redirection avec OTP Simuler une première connexion

Note : En environnement réel, la redirection viendrait du Portail Orange après une authentification MFA réussie.
